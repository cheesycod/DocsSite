you can follow the usage guides and examples below which should be more then enough to 
get you started xD

> NOTE: For posting stats you can send up to `3` requests every `5 Minutes`

---

## Javascript Usage

```js:title=POST
const fetch = require("node-fetch")
fetch(`https://api.infinitybotlist.com/bots/stats`, {
            method: "POST",
            headers: {
                "authorization": "api-key-here",
                "Content-Type": "application/json"
            },
            body: JSON.stringify({
                servers: 100,
                shards: 69
            })
        }).then(async res => console.log(await res.json()))
```

```js:title=GET
const fetch = require("node-fetch")
fetch(`https://api.infinitybotlist.com/bot/:botID`, {
            method: "GET",
            headers: {
                "Content-Type": "application/json"
            }
        }).then(async res => console.log(await res.json()));
```

## Python Usage

```python:title=POST
import aiohttp

# In a async function
async def post_stats():
    async with aiohttp.ClientSession() as sess:
        headers= {
            "authorization": "Your api token",       
        }
        payload= {
            "servers": len(bot.guilds) or "0", # Change this if you use custom clustering
            "shards": bot.shard_count or "0"   # Change this if you use custom clustering
        }
        await sess.post(f"https://api.infinitybotlist.com/bots/stats", headers=headers, json=payload)
```
